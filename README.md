## Introduction
This project analyzes the Stack Overflow Developer Survey for 2025 to explore popular programming languages/operating systems/ages/databases and more.

## Files
- `notebook.ipynb` : Notebook have main code and data analysis
- `survey_results_public.csv`: Download it [From here](https://survey.stackoverflow.co/2025/)
- `survey_results_schema.csv`: Schema for the survey questions.

## Libraries Used
1. **pandas**: For data manipulation and analysis.
2. **matplotlib**: For creating visualizations and plots.

## Results
- The most popular programming languages are HTML/CSS, JavaScript, and SQL.
- The most operating systems used are Windows , MacOS , and Ubuntu Linux.
- RDBMS still the best in databases field and users using SQL more than NoSQL , The most DBMS used are PostgreSQL , MySQL and SQLite.
